Description
-----------

EasyConfig is a Windows based utility that will allow you to quickly 
assign an IP address and easily configure your IOLAN DS.

------------
Installation
------------

To install EasyConfig perform the following steps:

	1. Copy the easyconfig.exe file to your PC. 
	2. Run easyconfig.exe. This will launch the EasyConfig  
	    program.
	3. Follow the simple configuration steps to configure your IOLAN DS.
