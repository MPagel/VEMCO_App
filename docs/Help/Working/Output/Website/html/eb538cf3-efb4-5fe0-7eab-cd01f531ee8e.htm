Events generated by the serial port manager.
Classes
  ClassDescription
ServerException
Server Exceptions are generated by the Serial Port Servicer when a fault mode is entered.
ServerRealTimeEvents
This provides a convenient base class for events generated by the Serial Port Manager.
ServerStartUp
ServerStartUp fires when... well the server starts up and is ready to look for VEMCO receivers attached via a serial port.
ServerStop
ServerStop fires when the server is no longer looking for receivers but may still have configured receivers that have not been removed and may still be reading data.
ServerStopped
This event fires when the server has stopped listening for new receivers and has also removed all configured receivers from the service so no more data is being read from the hardware.
