The Encoder builds commands as requested by the Receiver (possibly through public write() methods), by looking up the requested command in the configuration file. The configuration file specifies the format and the encoder uses String.Format to build the string that is finally returned. Before returning the string, however, it is verified as conforming to the config's spec.
